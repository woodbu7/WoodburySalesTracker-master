Title: The Sales Tracker
Description: This application uses the MVC design patterns and focuses on a clear separation of concerns. The salesperson can setup their account, and using CSV to implement persistence, the user can save and load their account info. The application will help the user keep track of their travels along with their sales and purchases of exercise equipments. 
Application Type: Console
